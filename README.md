# TakeMyApproval

A GitHub App built with [Probot](https://github.com/probot/probot) that automatically adds a GIF to your PR every time it gets an approval ✅

## Setup

```sh
# Install dependencies
npm install

# Compile typescript
npm run build

# Run the bot
npm start
```
